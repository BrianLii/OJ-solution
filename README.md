# UVa Online Judge Solutions

This repository contains my solutions to problems from the UVa Online Judge. Each solution under the `UVA/` directory has been Accepted by the judge. Feel free to view them if you need help or inspiration.

## Contributing & Feedback

If you find any bugs in the solutions or have suggestions for refactoring, feel free to open an issue to report problems.
